# 🏝️ 卡坦岛 (Catan)

单文件 HTML 实现的卡坦岛桌游，可直接部署到 GitHub Pages。

## 功能

- **标准规则**: 19 块六角地块、资源采集、建造、交易、发展卡
- **2-4 人本地多人** (热座模式)
- **完整游戏流程**: 布局阶段 → 掷骰子 → 采集资源 → 交易 → 建造 → 胜利
- **强盗系统**: 掷出 7 移动强盗，超过 7 张资源需丢弃
- **发展卡**: 骑士、胜利点、修路、丰年、垄断
- **港口交易**: 4:1 / 3:1 / 2:1 专属港口
- **最长道路** & **最大骑士团** 奖励
- **中文界面**

## 使用

直接在浏览器中打开 `index.html`，或部署到任何静态托管服务。

```bash
open index.html
```

## 部署到 GitHub Pages

1. 将此目录推送到 GitHub 仓库
2. Settings → Pages → 选择分支 → Save
3. 访问 `https://<username>.github.io/<repo>/`

## 操作说明

| 操作 | 说明 |
|------|------|
| 左键点击 | 放置建筑/道路 |
| 右键点击 | 取消当前操作 |
| 掷骰子 | 每回合开始时点击 |
| 交易 | 与银行或其他玩家交换资源 |
| 打卡 | 使用发展卡 |

## 技术

- 纯 HTML + CSS + JavaScript，无外部依赖
- Canvas 渲染六角地图
- 响应式布局
